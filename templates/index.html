Setup Guide — Multi-Output Audio on Windows

Windows only routes audio to one output by default. To send audio to multiple speakers simultaneously, use one of these methods:

Method 1: Stereo Mix (Built-in)

  1. Right-click the speaker icon in the taskbar → Sounds
  2. Go to the Recording tab
  3. Right-click and enable Show Disabled Devices
  4. Enable Stereo Mix
  5. Right-click Stereo Mix → Properties → Listen tab
  6. Check Listen to this device and select your secondary speaker
  7. Click Apply — audio now plays on both outputs

Method 2: Voicemeeter (Third-Party)

  1. Download and install Voicemeeter (free)
  2. Set Voicemeeter as your default playback device
  3. In Voicemeeter, assign multiple hardware outputs (A1, A2, etc.) to your Bluetooth speakers
  4. Audio is now routed to all assigned outputs

Once multi-output is configured, this mixer controls the volume balance between your speakers.
