        Setup Guide

        Windows only routes audio to one output by default. To send audio to multiple speakers:

        Method 1: Stereo Mix (Built-in)

        1. Right-click speaker icon → Sounds
        2. Recording tab → right-click → Show Disabled Devices
        3. Enable Stereo Mix
        4. Properties → Listen → check Listen to this device → select secondary speaker

        Method 2: Voicemeeter (Third-Party)

        1. Install Voicemeeter (free) → set as default playback
        2. Assign hardware outputs (A1, A2) to your speakers

        Keyboard Shortcuts

        • Arrow keys Move control point
        • Shift+Arrows Fine movement
        • Home Center control point
        • M Toggle mute on nearest speaker
        • 1-9 Load preset by number
        • Tab Cycle focus to next speaker
