Setup Guide

Method 1 — Windows "Listen to this device" (Stereo Mix)

This built-in Windows feature lets you mirror audio to a second output without any extra software.

  1. Open Control Panel > Sound (or run mmsys.cpl).
  2. Go to the Recording tab.
  3. Right-click in the empty area and enable Show Disabled Devices.
  4. Find Stereo Mix (or your sound card's loopback device), right-click it, and choose Enable.
  5. Double-click Stereo Mix to open its properties.
  6. Go to the Listen tab.
  7. Check Listen to this device.
  8. In the Playback through this device dropdown, select your Bluetooth speaker.
  9. Click OK. Audio from your default device will now also play on the Bluetooth speaker.
  10. Repeat for additional speakers, using one Stereo Mix instance per output.

Method 2 — Voicemeeter (third-party, free)

Voicemeeter is a virtual audio mixer that can route a single audio stream to multiple physical outputs with independent volume control.

  1. Download and install Voicemeeter (or Voicemeeter Banana for more channels) from vb-audio.com.
  2. Reboot after installation.
  3. Open Voicemeeter. Under Hardware Out (A1, A2, A3), assign each output to a different playback device (e.g., A1 = laptop speakers, A2 = Bluetooth speaker 1, A3 = Bluetooth speaker 2).
  4. In Windows Sound settings, set Voicemeeter Input as your default playback device.
  5. In Voicemeeter, make sure the input strip routes to A1, A2, and/or A3 by toggling the bus buttons.
  6. Adjust each output's fader to control volume independently, or use this mixer for spatial crossfading.
